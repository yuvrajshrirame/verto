# VERTO

> *"It doesn't matter where you start, it's how you progress from there."*

**VERTO** is a cyber-inspired focus ecosystem built for developers who value deep work, consistency, and clean aesthetics.

Turn every focus session into **XP**, sync your progress to **GitHub**, climb the ranks, and build a history of your discipline.

---

## ✦ Features

- 🎯 Precision focus timer
- ⚡ Real-time XP & rank progression
- 🔄 GitHub Markdown sync
- 🏆 Global leaderboard
- 🧠 Flexible category management
- 🌙 Minimal cyber-dark interface

---

## ⚡ XP System

```text
1 Minute = 10 XP
```

| Rank | XP |

## 🚀 Getting Started

```bash
git clone https://github.com/yuvrajshrirame/verto.git

cd verto
npm install
npm run dev
```

Create a `.env` file with your Firebase credentials before running the project.
